Privacy Policy for ROMPLFLOW

Effective date: May 23, 2026

ROMPLFLOW is a Chrome extension designed to help users manage AI generation workflows through a browser side panel.

Data Collection

ROMPLFLOW may process user activity data and website content only as required to provide the extension’s core functionality.

User activity data may include prompts, generation queue actions, generation history, task statuses, selected settings, and actions performed inside the ROMPLFLOW interface.

Website content may include data from supported Google Flow pages that is necessary to check connection status, detect whether a project is open, run user-requested generation workflow actions, and retrieve available generation results.

ROMPLFLOW stores extension settings, generation history, task statuses, prompts, and user-selected reference images locally in the user’s browser using Chrome storage.

ROMPLFLOW may connect to romplflow-access.romplflow.workers.dev to check access status, plan status, usage limits, or related account access information.

ROMPLFLOW does not sell user data.

ROMPLFLOW does not use user data for advertising.

ROMPLFLOW does not transfer user data to data brokers or information resellers.

Data Usage

Stored and processed data is used only to provide the extension’s core functionality, including:

  • saving extension settings;
  • displaying generation history;
  • managing the generation queue;
  • storing reference images selected by the user;
  • checking Google Flow connection and project status;
  • running user-requested generation workflow actions;
  • downloading generated results;
  • checking access status, plan status, and usage limits.

Third-Party Services

ROMPLFLOW is designed to work with Google Flow. ROMPLFLOW is not affiliated with Google.

When using Google Flow, data submitted to Google Flow is handled by Google according to Google’s own terms and privacy policies.

Data Storage

Most extension data is stored locally in the user’s browser. Users can clear history, cache, and stored references from within the extension interface.

Data Sharing

ROMPLFLOW does not sell, rent, or share user data with advertisers or data brokers.
